# Zeitdocs Codex Context Pack

This pack gives Codex the product, brand, content and visual context required to build the Zeitdocs public website and the Classify documentation using Astro and Starlight.

## How to use

1. Copy this full folder into the root of the website repository.
2. Keep `AGENTS.md` at the repository root.
3. Keep the `context/` and `prompts/` folders available for Codex.
4. Add real logos and screenshots under `public/assets/`.
5. Start Codex with the prompt in `prompts/01-initial-build-prompt.md`.

## Main objective

Build a clean, professional website for Zeitdocs and a documentation area for Classify.

The site should follow the visual direction of the included reference image:

`public/assets/references/zeitdocs-logo-exploration-style-reference.png`

Key style ideas:

- Soft light orange and light blue.
- Professional governance/documentation feel.
- White or off-white background.
- Thin dividers.
- Modular cards.
- Simple line icons.
- Clear product/documentation hierarchy.
- Calm enterprise SaaS visual tone.

## Important rule

Do not invent product functionality. If a product behavior is unclear, add a TODO comment and keep the documentation conservative.
