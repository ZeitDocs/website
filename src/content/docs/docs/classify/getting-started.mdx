This guide walks you through classification from the start, depending on your role.

## I am a page editor

If your space already has classification enabled, you can classify any page you
have permission to edit.

1. Open the page.
2. Look at the **byline** — the line near the top showing the author and date. You will see a **Classification** item with a coloured dot.
3. Click it to open the classification dialog.
4. Pick the classification level that matches the page's sensitivity.
5. Wait for the **"Classification saved ✓"** confirmation, then refresh the page.

That is all — the visual banner, label, and restrictions are applied automatically.

[GIF recommended: Open a Confluence page, select a classification level, and see the banner appear]

:::note
If you only see *"Classification is not enabled"*, the space has not been set up
yet. Ask your space administrator (see below).
:::

→ Full details: [Classifying a Page](/docs/classify/classifying-a-page/)

## I am a space administrator

You decide whether classification is available in your space and which levels
people can choose.

1. Open the byline classification dialog on any page, **or** go to your space's **Classification** settings page.
2. If classification is disabled, click **Enable classification**.
3. Choose your classification levels in one of two ways:
   - **Apply a global template** — pick a ready-made set of levels published by your Confluence admin (recommended for consistency).
   - **Add levels manually** — create your own levels with custom names, colours, banners, labels, and group restrictions.
4. Save. Page editors in your space can now classify pages.

→ Full details: [Space Settings](/docs/classify/space-settings/)

## I am a Confluence administrator

You manage classification across the whole site.

1. Go to **Apps → Classification Templates**.
2. Build or edit **global templates** — reusable level sets that space admins can apply.
3. Optionally use **Space Access Control** to decide which spaces are allowed to use classification at all.

→ Full details: [Global Templates](/docs/classify/global-templates/) and
[Space Access Control](/docs/classify/space-access-control/)

## How a page gets classified — the big picture

```text
Confluence admin            Space admin                 Page editor
─────────────────           ─────────────               ───────────
Creates templates  ──────▶  Applies a template  ──────▶  Picks a level
(once, site-wide)           to the space                 on each page
                            (or builds levels)                 │
                                                               ▼
                                          Classify applies banner + label +
                                          view/edit restrictions + report entry
```

:::caution
Do not rely on classification alone to protect sensitive content. Classify
applies real Confluence restrictions based on each level's groups, but you
should still review Confluence permissions when strict access control is
required.
:::
