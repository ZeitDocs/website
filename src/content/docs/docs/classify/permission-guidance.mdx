# Permission Guidance

Permission guidance helps users understand whether page access appears aligned with the selected classification level.

For example, a page marked **Confidential** may require a more careful permission review than a page marked **Internal**.

## Classification vs permissions

Classification and permissions are related, but they are not the same thing.

- Classification communicates how the content should be handled.
- Permissions control who can access the page.

:::caution
Do not rely on classification alone to protect sensitive content. Review Confluence permissions when access control is required.
:::
