# Global Templates

Global templates help admins standardize classification settings across spaces.

They are useful when an organization wants consistent classification levels, visual styles and permission guidance.

## When to use global templates

Use a global template when:

- Multiple spaces should share the same classification model.
- Admins want consistent names and descriptions.
- New spaces should start from an approved classification setup.
- Governance teams need reusable configuration.
