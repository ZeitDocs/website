# Space-Level Configuration

Space-level configuration lets a space use classification settings that match its local governance needs.

For example, a Finance space may need stricter levels and guidance than a general internal knowledge base.

## When to use it

Use space configuration when:

- A space has specific sensitivity requirements.
- A team needs a smaller set of classification levels.
- A space should use different visual styles from the global default.
- A space admin needs guidance that matches local processes.
